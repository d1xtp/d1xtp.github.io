Factorio Server Manager 🚀

Версия: 1.0.0

Менеджер серверов Factorio для удобного управления игровыми серверами.

⚠️ Внимание:

Для установки приложения ClickOnce может потребоваться:

  • Разрешить установку приложений из неизвестных источников
  • Нажать "Разрешить" при появлении предупреждения SmartScreen

📥 Установка

Нажмите кнопку ниже для установки приложения:

Установить Factorio Server Manager

Или скачайте и запустите setup.exe для установки

🔄 Обновления

Приложение будет автоматически проверять наличие обновлений при каждом запуске.

❓ Поддержка

Проблемы с установкой? Попробуйте:

  1. Обновить .NET Framework
  2. Отключить антивирус на время установки
  3. Запустить установку от имени администратора
